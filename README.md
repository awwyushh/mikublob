# MikuBlob

Track learning, one Blob at a time.

## Stack

- Next.js
- TypeScript
- Tailwind CSS

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Included in this UI scaffold

- Miku-inspired landing hero
- Calendar-first dashboard preview
- Day view and Blob entry form preview
- Search, tags, and stats panels
- Light and dark theme toggle with persistence
